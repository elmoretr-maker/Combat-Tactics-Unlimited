# CraftPix pack — environment

Place **generic map props** here when they are not part of the city-specific import.

## Prefer dedicated folders for the mega-set

- **City tiles** → `../city/` (PNG City, PNG City 2)
- **City buildings** → `../city_buildings/`
- **Boats** → `../boats/`
- **Shadows** → `../shadows/`

## This folder (plan)

- **Rural / misc tiles** — non-urban ground, props
- **Bridges** — if not filed under `city/`
- **Other structures** — bases, cover objects not sorted into `city_buildings/`

Optional subfolders: `tiles/`, `bridges/`, `props/` once you unpack.

See **`../README.md`** for the full pack map.
