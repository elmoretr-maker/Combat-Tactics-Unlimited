# CraftPix pack — fx

Place **game-ready VFX** you intend to load from code (explosions, fire loops, combined battle VFX).

## Related mega-set folders

- **`../smoke/`** — PNG smoke from the pack (plumes, haze)
- **`../effects/`** — CraftPix **Effects** folder mirror
- **`../hud/`** — HUD is UI, not world VFX

## This folder (plan)

- **Explosions** — impact / detonation animations staged for the engine
- **Fire** — burning, sustained fire
- **Smoke** — only if you duplicate or process from `../smoke/` for runtime

Subfolders like `explosions/`, `fire/` help keep paths stable for `craftpixHandler.js`.

See **`../README.md`** for the full pack map.
