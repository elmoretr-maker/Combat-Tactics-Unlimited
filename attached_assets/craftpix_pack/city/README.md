# City tile / backdrop art

## Expected contents

- **`PNG City`** — copy PNGs here (e.g. subfolder `city_1/` if you like).
- **`PNG City 2`** — second tileset variant (e.g. subfolder `city_2/`).

## Do not copy into the game tree

- **PSD City**, **PSD City 2** — Photoshop sources only; keep outside `craftpix_pack` or in a non-runtime archive.

Use this folder for ground tiles, roads, and large city ground layers. Put **individual building sprites** in `city_buildings/` if you want them separate from tile sheets.
